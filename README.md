# Famiglia-Bianchi
Projeto do curso de verão 2026, O projeto Famiglia Bianchi é um site institucional de uma pizzaria familiar fictícia, desenvolvido com HTML5 e CSS3. O foco está em UX/UI, navegação intuitiva e design inspirado na tradição italiana. O site apresenta cardápio, história e contato, com layout responsivo e elementos interativos em CSS.
